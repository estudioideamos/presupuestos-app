# Generador de presupuestos Ideamos

Aplicación web estática para crear presupuestos con la identidad visual de Ideamos, previsualizar seis páginas A4 y descargar el resultado en PDF.

## Uso

Abrir `index.html`, completar los campos del panel izquierdo y seleccionar **Descargar PDF**. Los cambios quedan guardados automáticamente en el navegador.

## Publicación

El proyecto no requiere compilación ni servidor. Puede publicarse directamente con GitHub Pages desde la rama principal.
